# Security Policy

## Reporting
Report vulnerabilities privately to repository maintainers before public disclosure.

## Scope
In scope:
- `src/ChainLocalizedRoutingHook.sol`
- `src/RoutingPolicyRegistry.sol`
- policy modules and supporting scripts

## Notes
- This system is not claimed attack-proof.
- Governance/admin permissions are a critical trust boundary.
- Deterministic policy logic is the intended security primitive.
