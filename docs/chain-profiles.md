# Chain Profiles

## Base
- Throughput-oriented.
- Higher swaps-per-block ceiling.
- Larger liquidity-relative max amount defaults.

## Optimism
- Stricter amount/rate controls.
- Conservative cooldown defaults.
- Optional fee adjustment path.

## Arbitrum
- Router allowlist-ready mode.
- Congestion-like dynamic fee adjustment path.
- Distinct policy outcomes for identical swap attempts.
